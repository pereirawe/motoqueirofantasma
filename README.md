🤝 Como Contribuir
------------------

Este é um projeto de código aberto e contribuições são muito bem-vindas! Para garantir a organização e a qualidade do código, seguimos o fluxo de **Pull Requests**.

### 🚀 Passo a Passo para Colaborar

Se você é novo no GitHub, siga estes comandos no seu terminal:

#### 1\. Faça um Fork do projeto

Clique no botão **Fork** no topo direito desta página. Isso criará uma cópia do repositório na sua conta.

#### 2\. Clone o seu Fork

Substitua SEU\_USUARIO pelo seu nome de usuário do GitHub:

#### 6\. Abra um Pull Request

1.  Vá até a página original do repositório no GitHub.
    
2.  Você verá um aviso sugerindo o **Compare & pull request**.
    
3.  Clique nele, descreva suas melhorias e envie.
    

### ⚠️ Regras Importantes

*   **Aprovação:** Todas as atualizações passam por uma revisão manual. Eu sou o único responsável por aprovar e mesclar (merge) o código na branch principal (main).
    
*   **Commits na Main:** Push direto para a branch main é bloqueado. Use sempre o fluxo de Pull Request acima.
    
*   **Sincronização:** Antes de começar uma nova alteração, certifique-se de que seu fork está atualizado com a main original para evitar conflitos.
    

### 💎 Dicas para Iniciantes

*   Mantenha seus Pull Requests focados em apenas uma melhoria por vez.
    
*   Se tiver dúvidas, abra uma **Issue** explicando o que deseja implementar antes de começar a codar.
    

**Obrigado por ajudar a melhorar este projeto!**
